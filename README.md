# TaskFlow - Project Management Tool

A full-stack project management application (Trello/Asana style) built as part of the **CodeAlpha Internship - Task 3**.

## Features
- User Registration & Login
- Create & Delete Projects
- Kanban Board (To Do / In Progress / Done)
- Create Tasks with Priority, Due Date & Assignee
- Move Tasks between columns
- Task Detail View
- Comment on Tasks
- Delete Tasks & Comments

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** SQLite
- **Auth:** JWT, bcryptjs

## How to Run
1. Clone the repository
2. Install dependencies:
   npm install
3. Start the server:
   node server.js
4. Open browser:
   http://localhost:3000
